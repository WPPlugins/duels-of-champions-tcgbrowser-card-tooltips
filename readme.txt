Transform Might&Magic Duel of Champions card names into links that show the card image when hovering over them.

== Description ==

The plugin adds a button in the visual post editor. To use it, just select the text that
represents a Duel of Champions card name, and click the button. It will turn the text into a
link to the card's page on http://mmdoc.tcgbrowser.com. Note that the tooltips will work only after
you publish or preview the post.

== Installation ==

Option 1:
1. Go to "Install Plugins" section of the Admin Panel, select "Add New" and then the "Upload" link
    to install the zip file.
2. Activate the plugin.
3. Use the "MMDoC Card" button in the editor.

Option 2:
Alternatively, you can search for "Duel of Champions TCGBrowser" from the Admin panel Plugins section, select this
plugin and click 'Install'.
